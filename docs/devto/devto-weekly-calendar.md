# AgentGovernance — Dev.to Weekly Calendar

**Cadence:** 1 post every **Tuesday** (publish by 10am PT).  
**Canonical:** always the matching `agentgovern.ai` guide (not Dev.to as primary).  
**After publish:** ledger row → `Posted`; add Dev.to URL; optional Hashnode cross-post (Dev.to canonical).

## Rules (same as SEO pages)

- One industry · one failure mode · one action
- Business / IT-admin voice — not "policy engine" jargon
- Link one guide + one CTA (`/agent-governance-demo` or waitlist)
- Tags: pick 4 from `ai, security, devops, microsoft, chatgpt, compliance, startup`
- Max ~1,100 words · `published: false` until you hit Publish on Dev.to

## Publish checklist (every Tuesday)

```
□ Read draft in docs/devto/ — tweak hook if needed
□ Confirm canonical_url matches live guide
□ Cover image: optional PNG 1000×420 (brand purple #6C5CE7 / dark bg) — or skip
□ Dev.to → New post → paste markdown → set canonical URL field
□ Tags: 4 max from list above
□ Publish → copy Dev.to URL to ledger
□ GSC: URL Inspection on canonical guide (not Dev.to) if guide impressions flat
□ Optional: Hashnode same day, canonical = Dev.to URL
□ Optional: LinkedIn 3-sentence summary + link to guide (not Dev.to)
```
